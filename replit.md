# AlgDevs-AI

A full-stack autonomous agent platform with Chat Mode, Plan Mode, and Agent Mode. Built for free-tier only — no credit card, no PC required (works on Android/Termux).

## Run & Operate

- `pnpm --filter @workspace/api-server run dev` — run the API server (port 5000 / proxied at /api)
- `pnpm --filter @workspace/algdevs-ai run dev` — run the frontend (proxied at /)
- `pnpm run typecheck` — full typecheck across all packages
- `pnpm run typecheck:libs` — rebuild lib declarations (run after changing lib/db or lib/api-spec)
- `pnpm run build` — typecheck + build all packages
- `pnpm --filter @workspace/api-spec run codegen` — regenerate API hooks and Zod schemas from OpenAPI spec
- `pnpm --filter @workspace/db run push` — push DB schema changes (dev only)
- Required env: `DATABASE_URL` — Postgres connection string (Replit built-in)

## Stack

- pnpm workspaces, Node.js 24, TypeScript 5.9
- API: Express 5 (port 8080, proxied at /api)
- Frontend: React + Vite (proxied at /)
- DB: PostgreSQL + Drizzle ORM
- Validation: Zod (`zod/v4`), `drizzle-zod`
- API codegen: Orval (from OpenAPI spec)
- AI: Groq SDK (primary, free tier) + Replit AI proxy (Gemini/OpenRouter/OpenAI/Anthropic, no key needed)
- Sandbox: E2B (free tier)
- Build: esbuild (CJS bundle)

## Where things live

- `lib/api-spec/openapi.yaml` — OpenAPI spec (source of truth for all API contracts)
- `lib/api-client-react/src/generated/api.ts` — generated React Query hooks
- `lib/api-zod/src/generated/api.ts` — generated Zod schemas
- `lib/db/src/schema/` — Drizzle ORM schema files (one per domain)
- `artifacts/api-server/src/routes/` — Express route handlers
- `artifacts/api-server/src/lib/ai.ts` — multi-provider AI abstraction
- `artifacts/algdevs-ai/src/pages/` — frontend pages

## Architecture decisions

- Contract-first: OpenAPI spec → Orval codegen → typed hooks + Zod validators on both ends
- Groq is the primary AI provider (free, fast); Replit AI proxy provides Gemini/OpenRouter as fallbacks — no API key needed for fallbacks
- E2B REST API used directly (not SDK) to keep bundle size low
- All mutations invalidate React Query cache automatically via Orval-generated hooks
- DB schema is one file per domain for maintainability

## Product

- **Dashboard**: Platform telemetry — active tasks, artifacts, sessions, events, provider health
- **Chat**: Multi-session AI chat with provider selection (Groq, Gemini, OpenRouter, etc.)
- **Plans**: Create and execute multi-step agentic workflows with step tracking
- **Tasks**: Monitor, cancel, and retry agent tasks with status filtering
- **Artifacts**: Browse and manage outputs generated by agents
- **Workspaces**: Isolated project environments
- **Agents**: Spawn and manage Supervisor, Planner, Researcher, Coder, Reviewer agents
- **Sandbox**: Execute Python/JS/TypeScript/Bash/Ruby code in E2B isolated environment
- **Settings**: Configure AI providers, test health, view platform info

## User preferences

- Free-tier only: no credit card, no PC (Android/Termux user)
- Groq free tier + Replit AI proxy = zero AI cost
- E2B free tier for sandboxed code execution

## Gotchas

- After changing `lib/db` schema, run `pnpm run typecheck:libs` before leaf package typechecks
- After changing `lib/api-spec/openapi.yaml`, run `pnpm --filter @workspace/api-spec run codegen` and then `pnpm run typecheck:libs`
- API server routes must be wired in `artifacts/api-server/src/routes/index.ts`
- Use `eq` from drizzle-orm as a static import (never dynamic `require`)
